PREGUNTAS SOBRE CSS:

Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS?

Para mi punto de vista, aplicaria !important solo cuando quisiera probar con otros diferentes diseños, ya sea en tipografía, color de imagen, gif. ETC..Ya que este comando toma el valor que contenga esta palabra como prioridad sin importar el orden de cascada

Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?

Esto se debe a que los elementos de la imagen contienen 3 atributos: Fuente, text y estilo. También por lo que al escojer una imagen existe el riesgo que el contenido de la página Web se pierda en la imagen, así mismo como la imagen al expaderse se puede pixelear.

Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt?

Para este caso, deberemos tener presente en donde se vizualisara nuestra página, ya que los px al ser un poco más viejos pierden calidad, y los porcentajes son un poco mejores al momento de hacer grande la letra

¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?

Esto se debe a que si una página cuenta con muchos estilos en CSS puede afectar al tiempo de carga que esta tiene, por lo consiguiente al minimizarla puede cargar con mucho mayor rapidez
